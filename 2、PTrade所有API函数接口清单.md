
# 设置函数清单

## set_universe
- **名称**: 设置股票池
- **接口说明**: 该函数用于设置或更新此策略要操作的股票池。

## set_benchmark
- **名称**: 设置基准
- **接口说明**: 该函数用于设置策略的比较基准，前端展现的策略评价指标都基于此处设置的基准标的。

## set_commission
- **名称**: 设置佣金费
- **接口说明**: 该函数用于设置佣金费率。

## set_fixed_slippage
- **名称**: 设置固定滑点
- **接口说明**: 该函数用于设置固定滑点，滑点在真实交易场景是不可避免的，因此回测中设置合理的滑点有利于让回测逼近真实场景。

## set_slippage
- **名称**: 设置滑点
- **接口说明**: 该函数用于设置滑点比例，滑点在真实交易场景是不可避免的，因此回测中设置合理的滑点有利于让回测逼近真实场景。

## set_volume_ratio
- **名称**: 设置成交比例
- **接口说明**: 该函数用于设置回测中单笔委托的成交比例，使得盘口流动性方面的设置尽量逼近真实交易场景。

## set_limit_mode
- **名称**: 设置回测成交数量限制模式
- **接口说明**: 该函数用于设置回测的成交数量限制模式。对于月度调仓等低频策略，对流动性冲击不是很敏感，不做成交量限制可以让回测更加便捷。

## set_yesterday_position
- **名称**: 设置底仓
- **接口说明**: 该函数用于设置回测的初始底仓。

## set_parameters
- **名称**: 设置策略配置参数
- **接口说明**: 该函数用于设置策略中的配置参数。

## set_email_info
- **名称**: 设置邮件信息
- **接口说明**: 该函数用于设置邮件信息。
